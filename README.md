\#  ShopBot Automation Framework



\##  Introduction



ShopBot Automation Framework is a Selenium-Java based test automation framework designed to validate an e-commerce web application.

The framework automates end-to-end user flows including login, product selection, cart operations, checkout process, and logout.



This project is built using Page Object Model (POM) design pattern to ensure code reusability, maintainability, and scalability.



\---





\##  Tech Stack



\* Java

\* Selenium WebDriver

\* TestNG

\* Maven

\* WebDriverManager

\* Extent Reports

\* Apache POI (Excel handling)



\---







\##  Key Design Principles



\* Page Object Model (POM)

\* Reusable methods in BasePage

\* Data-driven testing

\* Clean and readable code
